Go To Code

InGroup Travel Management

inGroup focusses on group travel management. If you want to travel as a group, it is sometimes difficult because everyone has his own wish list of places to go. To make it easier we provide a platform in which you can create new trips, add other users and easily share spots you want to see. All users can see the spots and add their own. This project is in progress, we try to translate it into a fully working PWA and add more features in the future. Technologies used: Angular, TypeScript, SASS, JavaScript.


Go to Project

Niubee Project

Niubee aims to be an online platform for 'Niubees' (a person who recently moved to another city). Locals can register and share their own secrets (spots they like beside tourist spots), so Niubees can have a better, personal view of the new city they now live in. This project is purely made with Node.js, Express, JavaScript, and SASS/SCSS.
